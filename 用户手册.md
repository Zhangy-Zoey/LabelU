# LabelU Video 用户手册

从视频里切出片段、可选画面裁切，按类别保存到对应文件夹。  
**裁剪导出**不改动原片；**批量分类**会把整段视频**移动**到类别子目录。

---

## 1. 导入

- **打开文件夹 / 打开文件**：选视频或含视频的文件夹  
- 也可把文件/文件夹 **拖进窗口**  
- 左侧列表点视频即可编辑；勾选「只看未完成」可过滤已点过「完成」的条目  
- **已完成**的视频也可点开回看：时间轴上会显示各分类片段（点彩色段可预览）。取消勾选「只看未完成」即可看到它们  
- 重新打开已处理过的文件夹时，会根据类别子目录中的导出**文件名**（及时段/裁切后缀）自动复原分类；旧版旁路 `.labelu.json` 若仍存在也可读

---

## 2. 标准流程（一条视频）

1. **拖时间轴两端手柄**设入/出点（点时间轴空白处只移动白色播放头）  
2. 需要画面裁切时点「裁切」，拖框选定区域  
3. **Enter** 或点「保存片段」→ 输入类别 → 确认  
4. 可继续切下一段；全部切完点「完成」（或按 → 自动完成）  
5. **← / →** 切到上/下一个视频

保存结果在：`原视频所在目录 / 类别名 /`，文件名形如  
`源文件夹_源文件名_序号_s入点ms_e出点ms_类别.mp4`；若做了画面裁切，末尾再加  
`_c左_上_宽_高`（相对画面 0–1，万分比整数）。  
重开文件夹时**只靠文件名**复原时间轴与裁切，不再写 `.labelu.json`。  
点「完成」会在源文件名加上 `_done` 标记（例如 `a.mp4` → `a_done.mp4`），**不另建标记文件、不改片内容**；撤销完成则去掉该后缀。  
类别子目录里的裁剪片段不会再出现在左侧列表（避免与源片混淆）。

选区至少 **2 秒**，且只能落在未分类的灰色剩余段上。

---

## 3. 时间轴

| 操作 | 怎么做 |
|------|--------|
| 移动播放头 | 点/拖时间轴空白处（白色竖线） |
| 设选区 | 拖入点（蓝）/ 出点（黄）手柄 |
| 调入/出点 | 拖两端手柄；松手靠近标记时会吸附 |
| 播放选区 | **空格** |
| 循环播放 | 打开「循环开」后再空格 |
| 看附近帧 | 拖手柄时下方出现选帧条，点某一帧可精确定点 |
| 放大选区 | 「轴」选「选区」；要看全片选「全片」 |
| 打标记 | **M** 或点「标记」；点标记可删；**Alt+M** 清全部 |
| 看已分类段 | 点彩色段预览；**Delete** 删除该分类结果后再重切 |

倍速、步进（1/8/16/22）在时间轴下方切换。步进越大，方向键微调越细。

---

## 4. 画面

- 播放区 **滚轮**：缩放；放大后可拖移；双击复位  
- 「裁切」：拖框/四角；再点一次可取消  
- 已分类片段不能再裁切，需先 Delete 删除该结果

---

## 5. 批量分类

整段视频直接归类（不裁切）：

1. 左侧 **Ctrl+单击** 多选，或 **Shift+单击** 连选  
2. 填类别 →「批量分类」  
3. 视频文件会被**移动**到 `原目录/类别名/`（不是复制）  
4. 若选中项已在类别文件夹内（二次分类），会询问落点：  
   - **原目录对应类别**：`/素材库/走路/a.mp4` → `/素材库/新类别/a.mp4`  
   - **当前目录下新建类别**：→ `/素材库/走路/新类别/a.mp4`  
   - **自选目标文件夹**：直接移入所选目录（不再套一层类别名），并记住路径  
   - 可勾选「不再询问」；批量窗内点「更改」可恢复询问  
5. 失败项可「重试」；整批可用 **Ctrl+Z** 或顶栏「撤回批量」撤回（仅当次进程有效，重启后不可撤）

---

## 6. 撤回

**Ctrl+Z** 顺序：**保存的片段 → 批量分类 → 选区/标记**。  
批量分类也可用顶栏「撤回批量」。

---

## 7. 快捷键

| 键 | 作用 |
|----|------|
| Enter | 保存片段 |
| 空格 | 播放/暂停选区 |
| Ctrl+Z | 撤回 |
| ← / → | 上/下一个视频 |
| [ / ] | 当前播放头设为入/出点 |
| Shift+← / Shift+→ | 微调入点 / 出点 |
| Alt+← / Alt+→ | 按原片帧率逐帧走播放头 |
| M | 加标记 |
| Alt+M | 清全部标记 |
| Delete | 删除选中的已分类片段 |
| Esc | 关保存窗 / 退出编辑态 |
| Ctrl+滚轮 | 缩放左侧缩略图 |

微调模式下（拖过手柄后）：← / → 按当前「步进」移动入或出点。

macOS 上 Ctrl 操作也可用 Command。

---

## 8. 注意

- 未点「完成」就退出：原片仍在；下次打开可恢复或丢弃本次已导出片段  
- 安装包未签名时：Windows 可能提示未知发布者（选「仍要运行」）；macOS 可能被 Gatekeeper 拦截（可右键「打开」）  
- 当前 macOS 安装包仅支持 Apple Silicon（arm64）  
- 处理中不要关软件；长时间导出可点取消（已完成的批量项仍可撤回）
