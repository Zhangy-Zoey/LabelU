# LabelU Video — 功能清单与快捷键

Windows / macOS 桌面端：从视频中切出片段（可选画面裁切），或导入图片裁切保存，按类别导出；也可整段/整图批量归类。  
快捷键按 **Windows（Ctrl）** 书写；macOS 上 **Ctrl 可用 Command**。

---

## 一、主要功能

### 1. 导入与列表

- 打开文件夹 / 打开文件，或拖入窗口；递归加载常见**视频与图片**（jpg/png/webp/bmp/gif 等）
- 若目录**同时含视频与图片**：弹窗询问「只加载视频 / 只加载图片 / 都加载」；选都加载后，媒体列表旁有「全部 / 仅视频 / 仅图片」筛选
- 左侧缩略图列表；角上勾选、小窗预览（视频）、预览进度条
- 「只看未完成」过滤；「完成」在源文件名加 `_done` 标记（不另建文件），不改片内容
- 左右分栏可拖；拖到边缘可隐藏一侧（分隔条双击还原）
- 裁剪导出的片段/图片不会进列表；整段批量归入类别目录的会标「已归类」
- 对已归类项二次分类：可选原目录对应类别 / 当前目录下新建类别 / 自选最终目录；支持「不再询问」与记忆目标路径
- 异常自动写入单一 `exceptions.log`；顶栏「查看日志」打开该文件；版本更新后日志覆盖清空
- GitHub Releases 自动更新（各平台只下载本平台安装包）；新版本首次打开弹出更新内容

### 2. 时间轴剪辑

- 拖入/出点手柄设选区；点时间轴空白处移动白色播放头（非设选区）
- 拖入/出点手柄；松手靠近标记可吸附
- 选区至少 **2 秒**，且须落在未分类剩余段内
- 空格播放选区；「循环开/关」（默认开）
- 「轴」：全片 / 选区放大
- 倍速 0.5 / 1 / 2 / 4；步进 1 / 8 / 16 / 22
- 播放头白线可拖动预览
- **M** 打标记；点标记可删；**Alt+M** 清全部

### 3. 选帧与微调

- 拖入/出点后出现选帧条；点某一帧定格（不自动播）
- 选帧区高度可拖大；蓝/黄框对应入/出点

### 4. 画面

- 播放区滚轮缩放；放大后可拖移；双击复位
- 「画面裁切」：进入后**不预设框**，在画面上按住拖拽框选；松手后可拖动/四角微调；再点可取消
- 已分类片段不可再裁切，需先删除该分类结果

### 5. 分类保存（核心）

- Enter 或「保存片段 / 保存图片」→ 选/新建类别 → 确认
- 类别为分组平铺标签，可手动加自定义标签
- 导出到 `原目录/类别名/`；文件名含源信息、序号、入出点、类别；有裁切再带 `_c…`
- **图片**：无时间轴，画面裁切后保存（可多次裁切到不同类别）；命名规则与视频片段一致（扩展名为 jpg/png/webp）
- **不写回、不覆盖原片**；重开靠文件名复原时间轴与裁切
- 彩色分类段可点预览；Delete / Backspace 删除后可重切
- 点「完成」，或用 ←/→ 切走时若已有分类会静默自动完成

### 6. 批量分类

- 勾选 / Ctrl+单击多选；Shift+单击从锚点连选中间全部
- 整段视频或整张图片**移动**到 `原目录/类别名/`（不是裁切导出）
- 失败可重试；Ctrl+Z 或「撤回批量」可撤（仅当次进程）

### 7. 撤回

Ctrl+Z 优先级：

1. 当前视频已保存的片段
2. 最近一次批量分类
3. 选区 / 标记（同一 UI 栈，后进先出）

## 二、快捷键

| 快捷键 | 作用 |
|--------|------|
| **Enter** | 打开「保存片段」；保存弹窗内确认导出 |
| **空格** | 选区播放 / 暂停（循环开时到出点重播） |
| **Ctrl+Z** | 撤回（见上：保存 → 批量 → 选区/标记） |
| **← / →** | 上一个 / 下一个视频（「只看未完成」时 → 会跳过已完成） |
| **微调中 ← / →** | 按「步进」移动当前正在调的入点或出点 |
| **Shift+← / Shift+→** | 非微调时：左调入点、右调出点（按「步进」） |
| **Alt+← / Alt+→** | 非微调时：按原片帧率逐帧移动播放头（不改选区） |
| **[ / ]** | 把当前播放头设为入点 / 出点 |
| **M** | 在播放头添加标记 |
| **Alt+M** | 清除全部标记 |
| **单击标记** | 删除该标记 |
| **Delete / Backspace** | 删除当前选中的已分类导出片段 |
| **Esc** | 关闭保存窗；或退出微调/裁切/分类预览等（不清除入出点数值） |
| **Ctrl+单击** | 多选 / 取消选中单个视频 |
| **Shift+单击** | 从锚点连选到此处（中间全部选中） |
| **Ctrl+滚轮** | 缩放左侧缩略图 |
| 播放区**滚轮** | 缩放画面（不用 Ctrl） |
| 缩略图角上**勾选** | 多选（与 Ctrl+单击同类） |

### 连选用法（Shift+单击）

1. 先点第一个视频（普通点或 Shift+点均可，作为锚点）
2. **按住 Shift**，再点另一个视频
3. 两端之间（含两端）全部选中；锚点保持在第一次，可继续 Shift+点更远项

### 方向键注意

- **进入微调**（拖过入/出点手柄）后：←/→ 只 nudge 当前端；此时 Shift/Alt+方向键也走同一逻辑，不会切视频或逐帧移播放头
- **选中已分类彩色段**预览时：Shift/Alt/[ / ] 无效；←/→ 仍可切视频

### 弹窗

保存、批量、确认、恢复会话等弹窗打开时，导航与剪辑类快捷键会暂时屏蔽，避免误操作。
